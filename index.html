ssh-wrappers

Two tiny POSIX shell wrappers around ssh that fix the most common day-to-day annoyances with key vs. password auth.

sshp

Force password auth — disable pubkey for one connection. Stops the agent from burning all your MaxAuthTries on keys before the server can ask for a password.
sshi

Use only explicitly configured identities (IdentitiesOnly=yes). Pin one key per host instead of letting the agent shotgun every loaded key.

Install

$curl -fsSL https://pacnpal.github.io/ssh-wrappers/install.sh | sh
  • Auto-detects your shell (zsh, bash, ksh) and writes to the right rc file
  • Idempotent — safe to re-run; never silently shadows existing definitions
  • --force to override, --uninstall to cleanly remove
  • POSIX shell only, no dependencies
